Welcome Dialog

Identifier

org.eclipse.ui.welcome

Description

This extension point provides for plugging in wizards to be shown as choices in the Welcome dialog box. These choices are added to the default choice to simply go to the workbench.

Markup

   <!ELEMENT extension (wizard*)>
   <!ATTLIST extension
     point CDATA #REQUIRED
     id    CDATA #IMPLIED
     name  CDATA #IMPLIED
   >

  • point - a unique extension point identifier
  • id - an optional extension identifier
  • name - an optional extension name

   <!ELEMENT wizard (EMPTY)>
   <!ATTLIST wizard
     id    CDATA #REQUIRED
     name  CDATA #REQUIRED
     class CDATA #REQUIRED
   >

  • id - a unique name that will be used to identify this Welcome wizard entry
  • name - a translatable name that will be used in the dialog box to represent this entry
  • class - a fully qualified name of a class that implements com.ibm.itp.ui.api.wizards.INewWizard interface.

Example

The following is an example of a Welcome extension definition: 

   <extension point="org.eclipse.ui.welcome"> 
      <wizard 
         id="com.xyz.quickWizard1" 
         name="XYZ Wizard" 
         class="com.xyz.wizards.quickWizard1"> 
      </wizard> 
   </extension>

API Information

The class attribute should represent a fully qualified name of a class that implements com.ibm.itp.ui.api.wizards.INewWizard interface.

Supplied Implementation

The workbench provides two choices in the Welcome dialog box:
  • Go to the workbench
  • Join a team
This list is subject to change depending on such things as installation options and product version. The end user product will typically include more choices contributed by plug-ins.
