Import Wizard

Identifier

org.eclipse.ui.importWizards

Description

This extension point provides for plugging in alternative import engines. These engines will be shown as choices in the import dialog box. Wizards can be made unavailable for certain target objects (current selection) using optional selection subelements. For example, an import wizard can be made available only for objects of type IProject.

Markup

   <!ELEMENT extension (wizard*)>
   <!ATTLIST extension
     point CDATA #REQUIRED
     id    CDATA #IMPLIED
     name  CDATA #IMPLIED
   >

  • point - a fully qualified identifier of the target extension point
  • id - an optional identifier of the extension instance
  • name - an optional name of the extension instance

   <!ELEMENT wizard (description? , selection*)>
   <!ATTLIST wizard
     id    CDATA #REQUIRED
     name  CDATA #REQUIRED
     class CDATA #REQUIRED
     icon  CDATA #IMPLIED
   >

  • id - a unique name that will be used to identify this wizard
  • name - a translatable name that will be used in the dialog box to represent this wizard
  • class - a fully qualified name of the class that implements org.eclipse.ui.IImportWizard interface
  • icon - an URI of the icon that will be used alongside the wizard name in the import engine listing

   <!ELEMENT description (#CDATA)>

   <!ELEMENT selection (EMPTY)>
   <!ATTLIST selection
     name  CDATA #IMPLIED
     class CDATA #REQUIRED
   >

  • name - an optional name filter that will be run against the target object name in order to calculate if the wizard applies to it
  • class - a fully qualified name of the target object class for which the wizard applies to
