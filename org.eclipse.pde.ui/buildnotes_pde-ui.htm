What's new in this drop

In addition to the items listed in the 3.2M5 New and Noteworthy section:
  • Removed 2.1 backward compatibility for self-hosting.
  • New templates for the org.eclipse.ui.decorators  and org.eclipse.ui.importWizards extension points has been added.
  • Added a Reset button to the Target Platform preference page to make it easy to restore to the default target platform location.
